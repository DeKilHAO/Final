1.Move the RankingSystemForEPL folder to somewhere u like.
2.Use IntelliJ IDEA CE(Or other IDE like Eclipse/Netbeans) to open the RankingSystemForEPL project.
3.Run the file "test.java" under "test" folder. 

Note:The simulation result will be stored in "output.csv",the location is "~/RankingSystemForEPL/output.csv".
     And the final rank after simulation will be stored in "finalrank.csv",the location is "~/RankingSystemForEPL/finalrank.csv".
     ( "~" represent the location of where you put the RankingSystemForEPL project.)
